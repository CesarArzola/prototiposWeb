Diccionario Maestro de HTML

Introducción al lenguaje

El HTML estructura el contenido de una web. Por ejemplo, define títulos, párrafos, imágenes, enlaces y listas.
Piensa en él como el esqueleto de una página web; sin HTML, no habría forma de organizar la información que ves en internet.

Etiquetas de Formato

En HTML, hr sirve para poner una línea horizontal que separa secciones, y br se usa para hacer un salto de línea dentro de un texto sin crear un párrafo nuevo.
